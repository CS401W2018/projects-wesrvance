The form tag has an "action" attribute. What does it do?

The action attribute in the form tag specifies the URL where the form data will be sent for processing when the form is submitted. It determines the server endpoint that will handle the form submission.

The form tag has a "method" attribute. What does it do?

The method attribute in the form tag specifies how the form data is sent to the server, usually as either get or post.

What is the difference between a "POST" and a "GET"?

Get sends data as part of the URL, making it visible and suitable for retrieving information, while Post sends data in the request body, keeping it hidden and ideal for sending sensitive or large amounts of data securely.
