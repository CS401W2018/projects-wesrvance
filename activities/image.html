1. What kind of photos are good to encode using JPEG?

Photos with lots of colors and details (like landscapes, portraits, and photographs) because JPEG compresses well and keeps file sizes small.

2. What kind of photos are good to encode using GIF?

Simple images with fewer colors (like logos, icons, and animations) because GIF supports transparency and simple animations.

3. What is unique about the PNG format?

PNG supports transparency and uses lossless compression, which means it keeps all the image details without losing quality.

4.How do you resize your photo to the size it should be displayedUse photo editing software (like Photoshop or GIMP) to set the image dimensions to the exact size you need?

Use photo editing software (like Photoshop or GIMP) to set the image dimensions to the exact size you need.

5. Why should you resize photos using photo editing software instead of resizing it using CSS?

Resizing in photo editing software keeps the image quality high and reduces file size, while CSS just changes how the image is displayed, which can lead to poor quality and larger file sizes.
